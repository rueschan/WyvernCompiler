Included in this release:

   * Lexical analysis
    
To build, at the terminal type:

    make
   
To run, type:

    ./wyvern.exe <file_name>
    
Where <file_name> is the name of a Wyvern source file. You can try with
these files:

   * 001_ultimate.wyvern
   * 002_binary.wyvern
   * 003_palindrome.wyvern
   * 004_factorial.wyvern
   * 005_arrays.wyvern
   * 006_next_day.wyvern
   * 007_literals.wyvern
